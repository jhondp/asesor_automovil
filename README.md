# Asesor de automóviles

## Descripción
No tengo ningún conocimiento sobre vehículos y últimamente estoy pensando en comprarme uno, pero no sé cuál es coger ni cuál tiene la mejor relación 
"Costo/Calidad". Adicionalmente, no tengo ningún amigo mecánico, así que ¿Qué puedo hacer?

Así que decidí crear este pequeño proyecto que tiene como objetivo generar una base de datos de vehículos y determinar el ranking de los mejores vehículos que hay 
en el mercado chileno
